# 💰 Expense Tracker

A simple and user-friendly web application to track daily expenses, manage monthly budgets, and analyze spending habits.

This project allows users to record expenses, categorize them, monitor their monthly budget, and generate reports to better understand where their money is being spent.

---

## 🚀 Features

### 🔐 Authentication

* Secure login and logout system
* User profile management

### ➕ Add Expenses

* Add new expenses easily
* Include:

  * Amount
  * Category
  * Date
  * Optional notes

### 📋 Expense List

* View all recorded expenses
* Organized by date
* Options to:

  * Edit expenses
  * Delete expenses

### 🔎 Filters

Filter your expenses using:

* Category
* Start date
* End date

### 💰 Budget Management

* Set a **monthly budget**
* Choose your **currency**
* Automatically calculate:

  * Total spent this month
  * Remaining budget

### 📊 Reports & Analytics

* Visual charts to analyze spending
* Expenses grouped by category
* Monthly insights

### 📥 Export Reports

Download reports in:

* **Excel format**
* **PDF format**

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/malik-tanveer/expense-tracker.git
```

Go to project folder:

```bash
cd expense-tracker
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

## 🌐 Deployment

This project is deployed using **Firebase Hosting**.

To deploy:

```bash
firebase deploy
```
